# Audiophile Ace

A multifunctional hi-fi audio tool. Bit-perfect playback, advanced DSP, spectral analysis, and more — for Desktop, Linux, and Android.

## Stack

- **Frontend** — Next.js 15 + TypeScript + Tailwind CSS
- **Audio Engine** — Pure C++17
- **Platform** — Tauri v2
- **Monorepo** — pnpm + Turborepo

## Dev

```bash
pnpm install

pnpm desktop:dev   # Windows / macOS
pnpm linux:dev     # Linux
pnpm android:dev   # Android
```
